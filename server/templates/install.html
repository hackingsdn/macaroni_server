Macaroni Project

This page should help you get the Macaroni Extension installed.

Download Macaroni Extension

To begin, you'll need a copy of the extension plugin.

Install Macaroni Extension

Point your browser to chrome://extensions. Then, drag the extension .crx file into your the Extensions tab window. Sometimes Chrome will give a warning here! Just try again and it should install.

Configure

Once installed, you should see an Options link in the Macaroni extension sub-section (found in "chrome://extensions"). On that page, you'll need to insert your API key.

Test

Head over to VirusTotal and try it out!
